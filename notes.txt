------------------------------ Atalhos Eclipse ------------------------------

alt shift a - selecionar várias linhas de uma vez para alterar
alt shift d - rodar classe atual em debug
alt shift r - refactor rename
alt shift x - rodar classe atual
alt up/down - mover linha
ctrl 1 - sugestões
ctrl 3 - quick access
ctrl alt up/down - copiar linha
ctrl d - deletar linha
ctrl f11 - rodar em debug
ctrl h - buscar string nos arquivos do projeto 
ctrl l - ir para uma linha do arquivo
ctrl m - maximizar/minimizar aba
ctrl n - new
ctrl o - procura atributos e métodos naquela classe
ctrl pageUp/pageDown - tab entre as abas abertas 
ctrl shift c - comentar bloco
ctrl shift f - identar código
ctrl shift f11 - apertar botão "run" do Eclipse
ctrl shift g - ver quem está chamando o método 
ctrl shift o - import
ctrl shift r - procurar resources
ctrl shift t - procurar classes
f3 - entrar em um método/classe
